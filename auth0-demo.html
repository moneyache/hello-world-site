Auth0 认证流程演示

点击登录后，浏览器会跳转到 Auth0 托管的登录页面完成认证，
认证成功后自动跳回本页并展示用户信息。

第一步：注册 Auth0 账号，创建 Application

在 auth0.com 免费注册，创建一个 Single Page Application 类型的应用， 拿到 domain 和 clientId 两个值。

第二步：配置回调地址

在 Auth0 后台把你的网站地址加到 Allowed Callback URLs、Allowed Logout URLs、 Allowed Web Origins 三个地方（比如 https://moneyache.github.io）。

第三步：引入 SDK，三行代码搞定

引入 @auth0/auth0-spa-js，调用 loginWithRedirect() 登录、 getUser() 获取用户信息、getTokenSilently() 拿 JWT token。
